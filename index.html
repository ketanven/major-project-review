Problem Statement

The Freelancer's Dilemma

⏱️

Inaccurate Time Tracking

Manual methods lead to missed billable hours and incorrect work records across multiple projects.

📄

Delayed Invoicing

Manual invoice creation increases calculation errors and makes tracking payment status difficult.

📉

Poor Financial Visibility

Spreadsheets and disconnected tools prevent real-time analysis of earnings and productivity trends.

"Freelancers juggling multiple clients struggle with scattered tools, handwritten notes, and spreadsheets—resulting in reduced productivity and missed revenue opportunities."

Project Objective

A Centralised Platform for Freelance Success.

  • ✦ Streamline client and project management into a single centralized platform.
  • ✦ Automate time tracking to ensure accurate billable hours and prevent revenue loss.
  • ✦ Generate professional, error-free invoices instantly with multi-currency support.
  • ✦ Provide powerful analytical insights into productivity and financial performance.

The Goal

To empower independent professionals by replacing fragmented workflows (spreadsheets, notebooks) with an integrated solution, allowing freelancers to focus on their craft whilst the system handles administration with precision.

System Architecture

Frontend Layer (ReactJS)

Provides interface for Freelancer/Admin, handles API requests, and stores secure JWT tokens.

Backend Layer (Django REST)

Handles robust business logic, RESTful API endpoints, and role-based access control.

Database Layer (MySQL)

Maintains relational integrity for users, clients, projects, tasks, and invoices.

User → React UI → REST API → MySQL → Response
